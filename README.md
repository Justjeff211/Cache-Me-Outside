
# Cache Me Outside - TryHackMe Write-Up

## Overview

This repository contains my write-up for the **Cache Me Outside** room on TryHackMe.

The challenge focuses on **Open Source Intelligence (OSINT)** techniques, requiring investigators to collect and correlate publicly available information from multiple sources to uncover details about a target.

Throughout the investigation, information was gathered from fitness applications, GitHub repositories, email metadata, social media platforms, and geolocation analysis to build a complete intelligence picture.

## Skills Demonstrated

* Open Source Intelligence (OSINT)
* Digital Footprint Analysis
* Username Enumeration
* GitHub Metadata Analysis
* Email Intelligence Gathering
* Social Media Investigation
* Photo Geolocation
* Critical Thinking & Investigation Methodology

## Key Takeaways

* Public Git metadata can expose sensitive information.
* Username reuse across platforms increases traceability.
* Social media posts can reveal location data unintentionally.
* Small pieces of information can be combined to create powerful intelligence findings.
